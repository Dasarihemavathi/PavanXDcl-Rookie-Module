# 🚀 PavanX Pro — DSA Rookie Module

A beginner-friendly **DSA (Data Structures & Algorithms) Learning Platform** built with React and JavaScript. This platform helps freshers and beginners learn DSA concepts through structured, module-wise content.

🌐 **Live Demo:** [pavan-x-dcl-rookie-module.vercel.app](https://pavan-x-dcl-rookie-module.vercel.app)

---

## 📌 About the Project

PavanX Pro DSA Rookie Module is an interactive 8-module learning platform designed for students and beginners who want to build a strong foundation in Data Structures and Algorithms. Each module covers a specific DSA topic with clear explanations and structured content.

---

## ✨ Features

- 📚 8 structured DSA modules (Arrays, Strings, Sorting, and more)
- 🔄 Module-wise navigation with progress tracking
- 🎯 Beginner-friendly content and clean UI
- ⚡ Fast and responsive design
- 🌐 Deployed and accessible online

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Dasarihemavathi/PavanXDcl-Rookie-Module.git

# Navigate to project folder
cd PavanXDcl-Rookie-Module

# Install dependencies
npm install

# Run locally
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view in browser.

### Build for Production

```bash
npm run build
```
